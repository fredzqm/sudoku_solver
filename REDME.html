CSSE 413: Constraint Satisfaction Programming Assignment

Objectives

The purpose of this assignment is to gain first hand experience that one can write software that is better than the person who develops it; with fairly little effort.

Assignment

  • Implement and test a backtracking algorithm to solve Sudoku problems.
  • Implement the the MRV, degree, and Least constraining value heuristics.
  • Extra credit for any additional improvements you may make. Please have a look at our book for ideas. We only scratched the surface.
  • Work on the implementation of this assignment by yourself.
  • You may assume a 9x9 grid.
  • Your software needs to read input as follows. A 9x9 grid of numbers, corresponding to the 9x9 layout of a typical Sudoku game. The number zero (0) represents an empty space. Below is an example.
